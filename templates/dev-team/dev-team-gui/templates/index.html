▸ Team Mode vs Agent Direct

Team Mode routes to the team orchestrator and runs a full pipeline (e.g. dev plan runs PM → BA → Architect → Security → UX in sequence).

Agent Direct targets a single specialist — no orchestrator overhead. Best for targeted tasks where you already have context.

💡 Good task descriptions

✓ Specific domain and context
✓ What you have (existing artifacts)
✓ What you need (deliverable)
✓ Any constraints or requirements
✓ Target platform or audience

⚡ Context resolution order

1. --project → output/<name>/context.json
2. --context-file → explicit file path
3. --context → inline string
4. None → runs with task only
